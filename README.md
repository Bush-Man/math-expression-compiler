## Features:


- [x] **Parse Let Statements**  
  Example:  
  ```
    let answer = "200 * 7 + 5 - 6 / 2";
  ```

- [x] **Parse Binary Expressions**  
  Example: 
  ``` 
    200 * 7 + 5 - 6 / 2;
  ```
- [x] **Print Abstract Syntax Tree (AST)**


- [ ] **Parse Parenthesized Expressions**  
  Example:
  ```  
    200 * 7 +(5 - 6) / 2;
    let answer = "200 * (7 + 5) - 6 / 2";
  ```

- [ ] **Parse Functions**  
   Example: 
    ```
      func multiply(a, b, c) { 
        return a * b * c; 
      } 
    ```  

- [ ] **Solve Expressions**  
   
   Example:
   ```
      let a = 40; 
      let b = 30; 
      let c = 40 + 30;  
   ```

- [ ] **Error Reporting with Line and Column Numbers**
